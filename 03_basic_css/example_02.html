Let's use: Cascading Style Sheets

The <span> element is used as a generic inline element that can be used for anything. This element is typically used when there are no other inline elements that make sense to use. You see lots of these in the wild.

The <div> element is used as a generic block HTML element that can be used for anything. This element is typically used when there is no other block element that makes sense to use. You see lots of these in the wild.
